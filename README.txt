## Project Overview

This project aims to provide a user-friendly interface for managing payments and generating reports of Grave Site. It utilizes React as the frontend framework and incorporates various components for efficient data management and user interaction.

### Navbar

The project includes a responsive navbar that provides easy access to different sections:

- Dropdown options for maintain, payments, and reports
- Search option
- Settings
- Help
- User menu

### Table Information

The application displays a table with the following information:

- Receipt number
- Date paid
- Post/Unpost status (toggleable)
- Grave site
- Paid by
- Number of payments
- Total

By clicking on the status, users can toggle between post and unpost, with a confirmation asked and the total count of unpost statuses is dynamically updated.

### Search and Filter

The application offers a search functionality allowing users to search for specific payment records based on receipt number, grave site, and payer. Additionally, a filter option is available to refine the displayed data.

### Add Payment

A dedicated button is provided to add new payment records, making it convenient for users to input new information.

### Footer

The footer section of the application contains company information, including:

- About: A brief description of the company and its mission.
- Terms of Use: Guidelines and policies governing the usage of the application.
- Contact: Contact information for inquiries, support, and feedback.

## Installation

To run this project locally, please follow these steps:

1. Ensure that you have Node.js installed on your machine.
2. Clone this repository to your local system.
3. Navigate to the project directory using the command line/terminal.
4. Run the command `npm install` to install the project dependencies.
5. Once the installation is complete, run the command `npm start` to start the development server.
6. Open your web browser and access the application at `http://localhost:3000` .


In the project directory, you can run:

### `npm start`


## Usage

Upon launching the application, you will be presented with the dashboard containing the payment table. You can navigate through different sections using the navbar and interact with the table by toggling the post/unpost status, using the search and filter options, and adding new payments.
